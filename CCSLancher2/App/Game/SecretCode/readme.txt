Secret Code

<ゲームについて>

●ゲームシステム
モジュールを配置し迫り来る敵からコアを守るシンプルなタワーディフェンスゲームです。
コアを守りきる事ができたらゲームクリアになります。
敵がマップの一番奥まで辿り着くとコアがダメージを受け、0になるとゲームオーバーになります。
敵を倒すとメモリを取得できるので、それを使ってモジュールを充実させよう！

●操作方法
・左クリック: 決定

●モジュール紹介
・Proto: 必要メモリが最も少ないモジュール。攻撃力は低いが連射力が高め。
・Buster: 連射力は低いが攻撃力が高めのモジュール。
・Pulse: 連射力がとても高いモジュール。
・Shocker: 敵の防御力を無視してダメージを与える事ができるモジュール。
・Wall: 範囲内の敵の移動速度を低下させる事ができるモジュール。
・Parallel: 最大3体の敵に同時攻撃する事ができるモジュール。
・Tracer: 圧倒的な攻撃範囲と連射力を持つモジュール。
・Iris: 攻撃範囲、攻撃力、連射力を高い水準で備えるモジュール。必要メモリが非常に多い。

●その他
・難易度はNormalとHardの2種類が存在し、敵のステータスが変化します。
(いずれも全25Waveで、敵の数やモジュールの能力は同じです。)
・アチーブメントに書かれた各条件を満たすと、あるメッセージを見る事ができます。
ぜひ挑戦し、"Secret Code"を解読してみて下さい。
