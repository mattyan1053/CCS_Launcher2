<操作方法>
このゲームはテンキーを使用します。
テンキーがない方申し訳ありません。
画面とテンキーの位置を重ね合わせていただくとだいたいその通りに動きます(ゲームを始めるにはセレクト画面で6を押す等)
テンキーの1〜9以外は使用しません。
ESC or 2, 4, 6, 8同時押し →　強制終了
